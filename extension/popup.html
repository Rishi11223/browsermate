BrowseMate

Welcome to BrowseMate

Control Chrome from AI agents or scripts

1 Start the bridge server
Download and run the server:
browsermate-server.exe
Or if you have Node.js:
cd server && npm install && node server.js
2 Connect
Click Connect below. The dot turns green when connected.
3 Send commands
Use any HTTP client to control Chrome:
curl -X POST http://localhost:3001/navigate -d '{"url":"https://...'
⚡ Available endpoints
/navigate — Go to a URL
/click — Click an element
/type — Type into an input
/extract — Extract page data
/screenshot — Take a screenshot
/eval — Run JavaScript
